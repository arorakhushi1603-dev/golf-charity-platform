# Golf Charity Platform

A web application that allows users to:

- Create Account
- Login
- Select Charity
- Purchase Subscription
- Submit Golf Scores
- Participate in Monthly Draws
- Admin Dashboard

## Tech Stack

- HTML
- css
- JavaScript
- LocalStorage

## Features

- Authentication
- Dashboard
- Charity Selection
- Subscription Management
- Monthly Draw
- Admin Panel
